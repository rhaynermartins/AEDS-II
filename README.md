## 📚 Sobre

Este repositório reúne implementações e materiais produzidos ao longo da disciplina de **Algoritmos e Estruturas de Dados II (AEDS II)**. O conteúdo está organizado para apoiar o estudo de análise de algoritmos, ordenação, pesquisa, estruturas lineares, estruturas flexíveis e árvores.

O objetivo é manter um registro acadêmico claro das atividades práticas da disciplina, incluindo exercícios de laboratório, preparação para provas, trabalhos práticos e slides de apoio.

## 💻 Conteúdos Implementados

### 📌 Exercícios e Laboratórios

- Implementações em **C** e **Java** para fixação dos conteúdos da disciplina
- Exercícios de laboratório envolvendo estruturas, ponteiros, células e árvores
- Atividades de preparação para provas com problemas de lógica e estruturas de dados

### 📌 Trabalhos Práticos (TPs)

- **TP1**: ciframento, alteração aleatória, validação de strings, inversão, soma de dígitos, anagramas, substrings e recursividade
- **TP2**: modelagem de dados, leitura de CSV, pesquisa sequencial e binária, algoritmos de ordenação, lista, pilha e fila circular
- **TP3**: seleção, inserção, quicksort e heapsort parciais, lista flexível, pilha flexível, fila flexível, lista dupla, matriz dinâmica e árvore binária

### 📌 Estruturas e Algoritmos

- Análise de complexidade e somatórios
- Algoritmos de ordenação: inserção, seleção, merge sort, quicksort, heap sort e counting sort
- Algoritmos de pesquisa: sequencial e binária
- Estruturas lineares: listas, pilhas e filas
- Estruturas flexíveis com alocação dinâmica
- Matrizes dinâmicas com células encadeadas
- Árvores binárias e operações de inserção, pesquisa, caminhamento e remoção

## ▶️ Como Compilar e Executar

### Programas em C

```bash
gcc TPs/TP1/q4inversao.c -o q4inversao
./q4inversao
```

### Programas em Java

Compile e execute informando o diretório do arquivo no classpath:

```bash
javac TPs/TP3/q9matrizDinamica.java
java -cp TPs/TP3 q9matrizDinamica
```

Os exercícios que utilizam `MyIO` dependem da biblioteca em `MyIO/MyIO.java`. Nesses casos, confira se o arquivo do exercício está compatível com o pacote/import da biblioteca antes da compilação.

> Alguns trabalhos utilizam arquivos auxiliares, como `TPs/TP2/restaurantes.csv`. Nesses casos, execute o programa considerando o caminho esperado pelo código-fonte.
